# Bamboo Product Shopping Website (EcoKart)

A full-stack e-commerce website for bamboo products.

## Features
- User registration & login
- Wishlist
- Cart system
- Order placement
- Account settings
- PHP backend
- MySQL database

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Put project folder inside htdocs
3. Start Apache & MySQL
4. Import database file:
   database/bamboo_database.sql
5. Open browser:
   http://localhost/project-folder-name

## Database
Database backup included inside:
